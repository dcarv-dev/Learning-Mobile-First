Testando Media Queries

Media Queries são importantes aliadas para construir sites responsivos com uma boa UX. Elas permitem adaptar o conteúdo de uma página de acordo com as propriedades do dispositivo que está a acessando. Neste site, você notará que em cada dispositivo há um papel de parede próprio além de outras diferenças, para que a experiência do usuário seja satisfatória tanto num smartphone quanto em uma grande televisão.
